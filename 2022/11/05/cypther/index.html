    官网

    命令 作用
    CREATE 创建节点\关系\属性
    MATCH 检索节点\关系\属性
    RETURN 返回查询结果
    WHERE 提供过滤条件
    DELETE 删除节点\关系
    REMOVE 删除节点\关系的属性\标签
    ORDER BY 排序检索数据
    SET 添加或更新标签

    • 在没有属性的节点之间创建关系

    • 为节点或关系创建单个或多个标签
